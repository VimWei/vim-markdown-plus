INTRODUCTION~
    vim-markdown-plus adds extra features to [vim-markdown](https://github.com/tpope/vim-markdown), making markdown editing more pleasant and productive.

FEATURES~
    - Easy toggle for bold, italic, strikethrough, code
    - Link creation, navigation, removal
    - Checkbox management
    - Code block helpers
    - Formatting/rendering helpers
    - ...

USAGE~
    See mappings and commands provided by the plugin.
